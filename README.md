

OPIS PROJEKTU:
Aplikacja webowa ASP.NET Core MVC do zarządzania budżetem osobistym. 
Pozwala na rejestrację wydatków, kategoryzację oraz analizę danych 
finansowych za pomocą interaktywnych wykresów w Panelu Admina.

WYKORZYSTANE TECHNOLOGIE:
- Backend: C#, ASP.NET Core MVC, Entity Framework Core, Identity
- Frontend: HTML5, CSS3, Bootstrap 5, JavaScript, Chart.js
- Baza danych: Microsoft SQL Server

INSTRUKCJA URUCHOMIENIA:
1. Otwórz projekt w programie Visual Studio (klikając plik .sln).
2. Kliknij zielony przycisk uruchomienia "Start" (Uruchom / Play) 
   na górnym pasku programu.
3. Projekt automatycznie skonfiguruje bazę danych przy pierwszym 
   starcie i otworzy aplikację w przeglądarce internetowej.
